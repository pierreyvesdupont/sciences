Etude du polynôme de second degrés P(x) = ax² + bx + c


Soit P(x) un polynôme de second dégrès de la forme : P(x) = ax² + bx + c, avec :
a = a
b = b
c = c

1. Je cherche mettre P(x) sous la forme d'une identité remarquable :

Au choix :

Si P(x) peut s'ecrire sous la forme (Ax + B)², alors :
P(x) = (Ax + B)² = A²x² + 2ABx + B²
=>a = A²
b = 2AB
c = B²
=>A = sqrt(a)
B = sqrt(c)
b = 2sqrt(a)sqrt(c)

P(x) peut donc s'écrire :
P(x) = (sqrt(a) + sqrt(c))²


ou

Si P(x) peut s'ecrire sous la forme (Ax - B)², alors :
P(x) = A²x² - 2ABx + B²
=>a = A²
b = -2AB
c > 0 et c = B²
=>A = sqrt(a)
B = sqrt(c)
b = -2sqrt(a)sqrt(c)

P(x) peut donc s'écrire :
P(x) = (sqrt(a) - sqrt(c))²


ou

Si P(x) peut s'ecrire sous la forme (Ax + Bx)(Ax - Bx), alors :
P(x) = (Ax + Bx)(Ax - Bx) = A²x² - B²
=>a = A²
b = 0
c < 0 et abs(c) = B²

P(x) peut donc s'écrire :
P(x) = (sqrt(a) + sqrt(c))(sqrt(a) - sqrt(c))
